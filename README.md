# React Ts Setup

This is a boiler plate for typescript based react apps

## Installation

To install and run this app on your local machine, follow these steps:

1. Clone this repository to your local machine
2. Navigate to the project directory
3. Run `yarn` to install the necessary dependencies
4. Run `yarn dev` to start the app
5. Open your browser and navigate to `http://localhost:5173`

## Dependencies

This app was built using the following dependencies:

- TypeScript
- React
- React Router
- Axios | React Query
- Tailwind | Mantine
- Redux Toolkit
- Redux Saga
- Mantine
- Headless UI
- AOS
- React toastify
- Sweetalert
- React-ts-typewriter
